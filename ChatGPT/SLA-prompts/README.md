# Что такое "продакшн" и SLA
- **Продакшн (production)** — рабочая среда, где система используется реальными пользователями и обрабатывает реальные данные.
- **SLA (Service Level Agreement)** — соглашение об уровне сервиса: гарантии доступности, времени реакции и ответственности за сбои.


## Как этим пользоваться на практике ##

В начале задачи вставляешь один файл, например:  
Работаем по sla-apache.md
или прямо копипастим содержимое в промпт.

- Apache → sla-apache.md
- Joomla / PHP → sla-php.md
- CI / deploy → sla-bash.md
- C# Backend сервисы → sla-csharp.md
