## 1. Метапланирование промптов

## 2. Суть техники
- ChatGPT сам выбирает **стратегию мышления**.
- Снимает с пользователя необходимость «угадывать промпт».
- Повышает качество сложных решений.

## 3. Как она работает
- Сначала анализирует тип задачи.
- Выбирает подход: логика, креатив, анализ.
- Потом уже решает задачу по плану.

## 4. Когда применять
- Сложные и размытые задачи  
- Стратегия, консалтинг, исследования  
- Междисциплинарные запросы  

## 5. Когда не применять
- Простые вычисления  
- Чёткие инструкции  
- Повторяющиеся шаблонные задачи  

## 6. Примеры промптов (адаптированы под ChatGPT)

### Пример 1. Предварительный анализ
Проанализируй задачу и определи, какой подход лучше использовать.

### Пример 2. План перед решением
Сначала разработай план решения, затем приступай к выполнению.

### Пример 3. Полный цикл
Проанализируй задачу, выбери стратегию, реализуй её и оцени результат.

## 7. Моё мнение об эффективности
- **Очень высокая**
- Особенно полезна для опытных пользователей
- Отличный кандидат на промпт по умолчанию
