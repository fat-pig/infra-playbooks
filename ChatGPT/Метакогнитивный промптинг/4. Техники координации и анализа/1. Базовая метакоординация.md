# Метакоординация инструментов

## Суть
Модель сама решает:
- какие способы работы нужны,
- в каком порядке их применять,
- где проверять результат,
- когда менять план.

Пользователю не нужно вручную «переключать режимы мышления».

## Как работает
1. Разбивает задачу на части
2. Подбирает подходящий способ для каждой
3. Выстраивает порядок действий
4. Проверяет промежуточные результаты
5. Меняет план, если что-то пошло не так

## Когда применять
- сложные задачи из нескольких этапов
- проекты с разными типами работы (анализ, расчёты, идеи)
- ситуации, где ошибка на раннем этапе ломает всё дальше

## Когда не нужно
- простые одношаговые задачи
- тривиальные запросы (перевод, справка, арифметика)

## Мнение об эффективности
Очень высокая.  
Это базовая техника для **продакшн-использования ChatGPT**.  
Сильно снижает количество ошибок и «прыжков логики».

## Примеры промптов

### Пример 1. Базовая координация
Разбей задачу на этапы.
Для каждого этапа определи оптимальный способ работы (анализ, расчёт, генерация, проверка).
Выстрой последовательность и поясни, почему она выбрана.

Задача:
{{описание задачи}}

---

### Пример 2. Координация с контролем качества
Проанализируй задачу и составь план решения.
Для каждого шага:
- цель шага
- ожидаемый результат
- критерий проверки качества

Если результат шага не проходит проверку — скорректируй план.

---

### Пример 3. Адаптивная координация
Построй план решения задачи.
Во время рассуждений:
- отслеживай логические ошибки,
- отмечай неопределённости,
- при необходимости меняй порядок шагов.

Финально выдай:
- итоговое решение
- краткий отчёт о том, какие корректировки были сделаны.
