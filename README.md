# infra-playbooks

Этот репозиторий содержит полезные советы, находки и руководства по работе с инфраструктурой и использованием больших языковых моделей (LLM, таких как ChatGPT) в инженерных задачах. Основной фокус на обеспечении качества и безопасности решений в продакшн-среде, включая SLA (Service Level Agreement) промпты для различных технологий.

## Индекс файлов

### ChatGPT/
- **[how-to-use-llm-in-engineering-problems.md](ChatGPT/how-to-use-llm-in-engineering-problems.md)**: Руководство по эффективному использованию LLM в инженерных задачах, включая принципы работы, ускорение процессов и проверку ответов.
- **[llm-antipatterns.md](ChatGPT/llm-antipatterns.md)**: Описание типовых антипаттернов при работе с LLM, таких как абстрактные best practices, отсутствие отката и уверенность без проверок, с рекомендациями по правильному подходу.
- **[STOP-commands.md](ChatGPT/STOP-commands.md)**: Набор команд для мгновенной коррекции поведения LLM, когда модель уходит в предположения или абстракции.

### ChatGPT/SLA-prompts/
- **[README.md](ChatGPT/SLA-prompts/README.md)**: Объяснение понятий "продакшн" и SLA, а также инструкции по использованию SLA-промптов для разных технологий.
- **[sla-apache.md](ChatGPT/SLA-prompts/sla-apache.md)**: SLA-промпт для работы с Apache и .htaccess, включая требования к правилам rewrite, порядку выполнения и критическим зонам.
- **[sla-bash.md](ChatGPT/SLA-prompts/sla-bash.md)**: SLA-промпт для Bash/Shell скриптов, с акцентом на безопасность, идемпотентность и проверки.
- **[sla-csharp.md](ChatGPT/SLA-prompts/sla-csharp.md)**: SLA-промпт для C# и .NET, охватывающий асинхронность, потокобезопасность и жизненный цикл объектов.
- **[sla-php.md](ChatGPT/SLA-prompts/sla-php.md)**: SLA-промпт для PHP, с требованиями к версии, обработке ошибок и тестированию.
- **[SLA-prompt.md](ChatGPT/SLA-prompts/SLA-prompt.md)**: Общий SLA-промпт для ChatGPT в продакшн-режиме, подчеркивающий детерминированную логику и работу с неопределенностью.
